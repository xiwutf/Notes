# 08-Daily

## 用途

Daily 存放**每日笔记与周期复盘**——Daily Note（日）和 Weekly Review（周）。这是系统的时间轴，连接 Inbox 处理、项目推进、思考记录。

## 示例内容

```
08-Daily/
├── 2026-05-26.md      # 今日 Daily Note
├── 2026-05-25.md
└── Reviews/
    └── 2026-W21.md    # 第 21 周复盘
```

## 使用建议

1. **命名规范**：
   - 日笔记：`YYYY-MM-DD.md`
   - 周复盘：`Reviews/YYYY-Www.md`
2. **Calendar 插件**：点击日期自动创建 Daily Note
3. **每日流程**：
   - 晨：打开 Daily → 写今日重点 → 处理 Inbox
   - 晚：回顾完成度 → 记录思考 → 准备明日
4. **Weekly Review**：周日使用 [[07-Templates/Weekly Review Template|Weekly Review Template]]
5. **Dataview 聚合**：Dashboard 通过 Dataview 展示最近 Daily 内容
