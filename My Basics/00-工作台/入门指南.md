# 📘 人生操作系统 · 入门指南

> 3 分钟搞懂这套笔记怎么用 — 全中文说明

**看不懂英文？没关系。** 笔记顶部的英文属性（type、status 等）是系统内部用的，**你不用管**，只填下面的中文正文即可。

---

## 第一步：每天从哪开始？

打开 Obsidian 后，你会看到 **[[首页|工作台首页]]**。

每天只需做两件事：

1. 看「今日重点」，写今天最重要的一件事
2. 需要记东西时，用下表的入口

| 我想… | 怎么做 |
|--------|--------|
| **开始读一本新书** | `03-知识库/读书` → 复制 **`BOOK-示例`** → 重命名为 `BOOK-书名` |
| **某句话触动了我** | `03-知识库/触动` → 新建笔记 → 命名 `INS-主题` |
| **有个想法先记下来** | 打开 `01-收件箱`，写一行字 |
| **写今天的日记** | `Ctrl+P` → 搜「日记」→ 打开/创建今天的日记 |
| **看在读哪些书** | 打开 [[读书看板]] |

> 按钮没显示？请切换到 **阅读模式**，并确认 Dataview 插件已启用。或直接打开 [[快捷按钮]] 页面。

## 第二步：几种笔记分别是什么？

用**大白话**区分，不用背英文：

| 你看到的文件名 | 中文叫法 | 干什么用 | 举例 |
|----------------|----------|----------|------|
| `BOOK-书名.md` | **读书笔记** | 一本书一条，记进度和感受 | `BOOK-思考快与慢.md` |
| `INS-主题.md` | **触动笔记** | 某句话打动你，记下为什么 | `INS-痛苦是进化信号.md` |
| `KNW-概念.md` | **知识笔记** | 一个可复用的概念，用自己的话写 | `KNW-复利效应.md` |
| `THK-主题.md` | **思考笔记** | 你形成的判断和观点 | `THK-长期主义实践.md` |
| `PRJ-项目.md` | **项目笔记** | 要交付的事，有截止日期 | `PRJ-数字孪生平台.md` |
| `AST-标题.md` | **输出资产** | 对外发布的文章、演讲等 | `AST-工业AI实践.md` |
| `SYS-系统.md` | **人生主线** | 四大长期方向的总览 | `SYS-AI事业系统.md` |

### 读书时怎么记？（最常见）

```
1. 点「新建读书笔记」→ 得到 BOOK-xxx
2. 读的时候有句子打动你 → 点「新建触动笔记」→ 得到 INS-xxx
3. 书里有个概念想以后用 → 另建 KNW-xxx（知识笔记）
4. 想通了一个道理 → 另建 THK-xxx（思考笔记）
5. 想落实到工作里 → 在项目 PRJ-xxx 里加一条任务
```

**不用一次全做。** 至少：BOOK + 有触动就 INS。

## 第四步：笔记属性（顶部那块）怎么填？

创建笔记后，顶部可能显示这些属性：

| 属性名 | 中文意思 | 怎么填 |
|--------|----------|--------|
| `status` / 状态 | 当前状态 | 读书：「在读」「已读」「搁置」；触动：「待关联」「已关联」 |
| `author` / 作者 | 作者 | 书名对应的作者 |
| `domain` / 领域 | 领域 | 如：AI、心理、商业 |
| `started` / 开始 | 开始日期 | 一般自动生成，不用改 |
| `rating` / 评分 | 评分 | 读完后打 1-5 分 |

**type、tags、created** 等不用手动改，系统自动识别。

---

## 第五步：三条原则（记住就够）

1. **少摘抄，多理解** — 「我的理解」必须用自己的话写
2. **少收藏，多行动** — 每条笔记尽量有一个 `- [ ]` 待办
3. **少整理，多思考** — 链接 2 条相关笔记即可，不要追求完美

## 常见问题

**Q：模板在哪？我要手动建笔记吗？**  
A：不用。首页点按钮会自动建好。模板在 `07-模板`，不要直接在里面写。

**Q：BOOK、INS 这些前缀要删吗？**  
A：不要删。输入书名时只打「思考快与慢」，系统会自动变成 `BOOK-思考快与慢`。

**Q：和 ChatGPT 怎么配合？**  
A：复制 [[06-AI系统/提示词/系统提示词|ChatGPT 系统说明]] 到 ChatGPT，再把书中段落贴过去，让它帮你整理成 INS/KNW 草稿。

**Q：详细手册在哪？**  
A：[[认知系统说明书]]（完整版）· [[人生主线|人生主线地图]]
