# 05-Assets

## 用途

Assets 是**可对外输出的复利资产库**——文章草稿、演讲稿、代码片段、设计模板、个人品牌素材。Knowledge 是输入与内化，Assets 是输出与变现。

## 示例内容

| 资产 | 类型 | 状态 |
|------|------|------|
| [[AST-工业AI实践系列-01]] | 文章 | 草稿 |
| [[AST-数字孪生架构演讲稿]] | 演讲 | 已完成 |
| [[AST-个人IP定位声明]] | 品牌 | 定稿 |
| [[AST-开源工具模板]] | 代码 | 维护中 |

## 使用建议

1. **命名规范**：`AST-资产名称`
2. **来源追溯**：每条 Asset 标注来源 Knowledge 笔记（`来源:: [[KNW-xxx]]`）
3. **状态管理**：`草稿 → 审校 → 发布 → 归档`
4. **发布清单**：发布前检查——目标读者、核心观点、行动号召
5. **复利思维**：同一 Knowledge 可衍生多个 Assets（文章、视频脚本、演讲）
6. **定期盘点**：每季度统计已发布资产，评估个人 IP 积累进度

## 输出流水线

```
03-Knowledge（内化） → 05-Assets（加工） → 外部发布（博客/社媒/演讲）
```
