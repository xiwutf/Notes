## 思考辅助

### 深度思考引导

```
你是一位思维教练。我将分享一个想法/问题，请帮我：
1. 识别核心假设
2. 提出 3 个反面论证
3. 建议下一步行动

我的想法：{想法内容}
```

### 知识内化

```
我将粘贴一段内容，请帮我：
1. 用一句话概括核心观点
2. 列出 3 个关键概念
3. 给出 2 个应用场景
4. 提出 1 个我可能忽略的角度

内容：{粘贴内容}
```

### 读书笔记

```
你是我的读书助手。我使用 Obsidian 第二大脑系统，请按以下结构帮我整理读书笔记。

【系统上下文】
- BOOK 笔记：一本书一条，命名 BOOK-书名
- INS 笔记：认知触发点，命名 INS-主题（触动句 + 为什么 + 项目启发）
- KNW 笔记：可复用概念，必须有「我的理解」
- THK 笔记：形成的判断与原创思考
- 知识闭环：BOOK → INS → KNW → THK → PRJ → AST
- 三问自检：我真正理解了吗？能用自己的话解释吗？这改变了我的行动吗？

【当前输入】
- 书名：{书名}
- 作者：{作者}
- 当前章节/进度：{章节}
- 原始内容或我的草稿：{粘贴内容}

【请输出】
1. 章节一句话摘要
2. 作者核心主张（3 条以内）
3. 触动我的句子（原文 + 为什么触动，建议 INS-xxx 标题）
4. 我的理解（第一人称，禁止照搬）
5. 可拆分为 KNW 的概念（每个概念给建议标题 KNW-xxx）
6. 是否形成判断（是否值得建 THK-xxx）
7. 对当前项目的启发（建议链接 PRJ-xxx）
8. 1-2 个可行动项
9. 建议关联的 SYS 系统（AI事业 / 财富自由 / 个人IP / 健康）
```

### Insight 生成

```
你是我的 Insight 助手。Insight = 触动 + 启发，不是知识，不是深度思考，而是认知触发点。

【系统上下文】
- INS 笔记：记录触发句、为什么触动、对项目的启发
- 知识闭环：BOOK → INS → KNW → THK → PRJ → AST
- 原则：少摘抄多理解，少收藏多行动，少知识多决策

【当前输入】
- 来源类型：{书籍 / 文章 / 视频}
- 来源：{书名或链接}
- 触发句：{原文}
- 我的初感：{为什么触动我}
- 当前活跃项目：{PRJ-xxx 列表，可选}

【请输出】
1. INS 笔记完整草稿（含：来源、触发句、为什么记录、我的理解）
2. 建议 INS 标题：INS-xxx
3. **链接建议**（必须给出）：
   - 关联 KNW：[[KNW-xxx]] — 理由
   - 关联 THK：[[THK-xxx]] — 是否已形成判断？若否说明「暂不需要」
   - 关联 PRJ：[[PRJ-xxx]] — 对哪个项目有何启发
   - 来源 BOOK：[[BOOK-xxx]]
   - 关联 SYS：[[SYS-xxx]] — 属于哪条人生主线
4. 下一步行动（1-2 条，可勾选）
5. 三问自检提醒
```

## 知识整理

### 笔记结构化

```
请将以下内容整理为结构化笔记，包含：
- 一句话概括
- 核心内容（分点）
- 我的理解（引导我用自己的话表述）
- 关联概念（建议 3 个可能相关的主题）
- 应用场景

原始内容：{内容}
```

### MOC 生成

```
以下是关于「{主题}」的笔记列表：
{笔记标题列表}

请帮我：
1. 按子主题分类
2. 识别核心概念枢纽
3. 建议缺失的主题
4. 生成 MOC 索引结构
```

## 内容创作

### 文章大纲

```
基于以下知识笔记，生成一篇面向 {目标读者} 的文章大纲：
- 核心观点：{观点}
- 参考笔记：{笔记内容}
- 目标长度：{字数}
- 风格：{专业/通俗/故事化}
```

### 技术博客

```
你是一位 {领域} 专家。基于以下要点，撰写一篇技术博客：
- 主题：{主题}
- 要点：{要点列表}
- 目标读者：{读者画像}
- 要求：有代码示例、有架构图描述、有实践建议
```

## 项目辅助

### 项目规划

```
我要启动一个项目：{项目描述}

请帮我：
1. 明确成功标准（SMART）
2. 拆解为 3-5 个里程碑
3. 识别 top 3 风险
4. 建议第一周的行动项
```

### 技术方案评审

```
请评审以下技术方案，从架构、可行性、风险三个维度：
{方案内容}

请给出：
- 优点（2-3 点）
- 问题（2-3 点）
- 改进建议（2-3 点）
```

## 复盘辅助

### Weekly Review

```
以下是我本周的记录：
- 完成的任务：{列表}
- 项目进展：{列表}
- 新的思考：{列表}

请帮我：
1. 总结本周亮点
2. 识别未完成项的原因
3. 建议下周 Top 3 优先级
4. 是否有值得深化的知识主题
```
