# 04-Life

## 用途

Life 存放**个人生活与长期成长**相关内容——健康、关系、习惯、财务规划、人生决策。与专业领域知识分离，但同样遵循长期积累原则。

|------|------|
| [[LIFE-2026年度目标]] | 目标规划 |
| [[LIFE-健康习惯追踪]] | 习惯 |
| [[LIFE-财务复利计划]] | 财富积累 |
| [[LIFE-重要关系维护]] | 关系 |

## 使用建议

1. **命名规范**：`LIFE-主题`
2. **与 Daily 联动**：健康、习惯类目标写入 Daily Note 的「生活」区块
3. **季度回顾**：每季度检查 Life 目标与实际偏差
4. **隐私意识**：此目录可能含敏感信息，同步时注意加密
5. **少而精**：Life 笔记聚焦真正影响长期幸福的事项，不做流水账
