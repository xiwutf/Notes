# Books · 读书笔记

## 放什么？

一本书记一条笔记，文件名：`BOOK-书名.md`

**不要在这里手动建空白文件** — 去 [[00-工作台/首页|工作台]] 点 **📖 新建读书笔记**。

## 和其他笔记的区别

| 类型 | 文件名 | 干什么 |
|------|--------|--------|
| 读书笔记 | `BOOK-书名` | 整本书的进度、评价、行动 |
| 触动笔记 | `INS-主题` | 某句话打动你 |
| 知识笔记 | `KNW-概念` | 可复用的概念 |
| 思考笔记 | `THK-主题` | 你想通的道理 |

## 读书流程（简版）

```
新建 BOOK → 读时有触动 → 新建 INS → 有概念 → 新建 KNW → 想通了 → 新建 THK
```
