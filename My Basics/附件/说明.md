# 附件

## 用途

附件 是**多媒体与附件统一存储区**——图片、PDF、截图、导出文件、代码附件。保持笔记目录干净，所有引用通过 Obsidian 链接指向此处。

## 示例内容

```
附件/
├── images/
│   ├── 2026-05-architecture-diagram.png
│   └── digital-twin-screenshot.png
├── pdfs/
│   └── paper-rag-survey.pdf
└── exports/
    └── project-deliverable.zip
```

## 使用建议

1. **子目录分类**：按 `images/`、`pdfs/`、`exports/` 组织
2. **命名规范**：`YYYY-MM-描述.ext`，避免 `IMG_001.png`
3. **Obsidian 设置**：Settings → Files & Links → Default location for new attachments → `附件`
4. **定期清理**：每季度删除无引用的孤儿附件
5. **体积控制**：大文件（>10MB）考虑外部存储，笔记中只留链接

## 引用方式

```markdown
![[附件/images/2026-05-architecture-diagram.png]]
```
