# 00-Dashboard

## 用途

Dashboard 是整个认知操作系统的**指挥中枢**。所有日常工作的起点与终点都在这里——查看今日重点、追踪项目进度、进入知识网络、启动 AI 工作流。

不在这里存放具体内容，只做**聚合与导航**。

## 使用建议

1. **设为 Homepage**：安装 Homepage 插件后，将 `Home.md` 设为启动页
2. **每日 ritual**：早上打开 Dashboard → 查看今日重点 → 进入 Daily Note
3. **每周 ritual**：周日打开 Dashboard → 跳转 Weekly Review
4. **保持精简**：Dashboard 文件不超过 10 个，只做索引不做仓库
5. **定期维护**：每月检查链接是否失效、项目状态是否更新
