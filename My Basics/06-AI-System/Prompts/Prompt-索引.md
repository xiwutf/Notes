# Prompt 索引

> 按场景分类的 Prompt 模板库 — 经过验证、可复用、持续迭代

---

## 使用方式

1. 找到对应场景的 Prompt
2. 复制到 AI 对话中
3. 替换 `{变量}` 部分
4. 效果不佳时迭代并更新此库

---

## 思考辅助

### 深度思考引导

```
你是一位思维教练。我将分享一个想法/问题，请帮我：
1. 识别核心假设
2. 提出 3 个反面论证
3. 建议下一步行动

我的想法：{想法内容}
```

### 知识内化

```
我将粘贴一段内容，请帮我：
1. 用一句话概括核心观点
2. 列出 3 个关键概念
3. 给出 2 个应用场景
4. 提出 1 个我可能忽略的角度

内容：{粘贴内容}
```

## 知识整理

### 笔记结构化

```
请将以下内容整理为结构化笔记，包含：
- 一句话概括
- 核心内容（分点）
- 我的理解（引导我用自己的话表述）
- 关联概念（建议 3 个可能相关的主题）
- 应用场景

原始内容：{内容}
```

### MOC 生成

```
以下是关于「{主题}」的笔记列表：
{笔记标题列表}

请帮我：
1. 按子主题分类
2. 识别核心概念枢纽
3. 建议缺失的主题
4. 生成 MOC 索引结构
```

## 内容创作

### 文章大纲

```
基于以下知识笔记，生成一篇面向 {目标读者} 的文章大纲：
- 核心观点：{观点}
- 参考笔记：{笔记内容}
- 目标长度：{字数}
- 风格：{专业/通俗/故事化}
```

### 技术博客

```
你是一位 {领域} 专家。基于以下要点，撰写一篇技术博客：
- 主题：{主题}
- 要点：{要点列表}
- 目标读者：{读者画像}
- 要求：有代码示例、有架构图描述、有实践建议
```

## 项目辅助

### 项目规划

```
我要启动一个项目：{项目描述}

请帮我：
1. 明确成功标准（SMART）
2. 拆解为 3-5 个里程碑
3. 识别 top 3 风险
4. 建议第一周的行动项
```

### 技术方案评审

```
请评审以下技术方案，从架构、可行性、风险三个维度：
{方案内容}

请给出：
- 优点（2-3 点）
- 问题（2-3 点）
- 改进建议（2-3 点）
```

---

## 复盘辅助

### Weekly Review

```
以下是我本周的记录：
- 完成的任务：{列表}
- 项目进展：{列表}
- 新的思考：{列表}

请帮我：
1. 总结本周亮点
2. 识别未完成项的原因
3. 建议下周 Top 3 优先级
4. 是否有值得深化的知识主题
```
