# 06-AI-System

## 用途

AI-System 是认知操作系统的**增强引擎**——Prompt 库、AI 工作流、自动化脚本说明、与 Obsidian 联动的 AI 使用规范。

让 AI 不是外部工具，而是系统的一部分。

```
├── Workflows/         # AI 工作流（思考辅助、知识整理、内容生成）

└── README.md
```

## 使用建议

1. **Prompt 版本化**：每次优化 Prompt 保留历史版本，标注效果
2. **工作流标准化**：重复 3 次以上的 AI 操作，固化为 Workflow
3. **输入输出规范**：Workflow 明确输入（哪类笔记）和输出（存到哪里）
4. **与 Knowledge 联动**：AI 辅助产出的知识，经人工审核后存入
5. **定期迭代**：每月回顾 Prompt 效果，淘汰低效的
