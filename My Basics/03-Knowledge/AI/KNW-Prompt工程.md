# KNW-Prompt工程

## 一句话

Prompt 工程是通过结构化指令引导 LLM 输出预期结果的实践方法论。

## 核心内容

- **角色设定**：明确 AI 扮演的专家角色
- **任务描述**：清晰、具体、无歧义
- **输出格式**：指定结构、长度、风格
- **Few-shot 示例**：提供输入输出样例
- **思维链（CoT）**：引导逐步推理

## 我的理解

好的 Prompt 不是「魔法咒语」，而是「精确的需求文档」。本系统的 Prompt 库存储经过验证的模板，而非临时拼凑。

## 应用场景

- 所有 AI 辅助工作流
- 内容创作
- 代码生成与审查
